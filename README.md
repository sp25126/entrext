# ENT R E X T

### *The Indestructible Audience Experience Platform*

**Entrext** is a high-fidelity audit and feedback engine designed for the next generation of web experience teams. It features an "indestructible" proxy tunnel that enables realtime, synchronized auditing of high-end portfolios, 3D/WebGL experiences, and secure web applications.

---

## 🛡️ The Architecture of Resilience

Unlike traditional annotation tools, Entrext is built to handle the "un-proxifiable." 

- **Indestructible Proxy Tunnel**: A hardened FastAPI-driven proxy that automatically detects and fallbacks for WebGL/3D environments.
- **Defensive Selector Engine**: A crash-proof element selection algorithm that identifies targets across complex Shadow DOMs, SVGs, and dynamic canvas layers.
- **Realtime Peer-Sync**: Low-latency WebSocket orchestration for live, collaborative audit sessions.
- **AI-Ready Exports**: Context-rich markdown data structures designed for seamless integration with LLMs and project management tools.

## 🚀 MVP Capabilities

- [x] **Indestructible Routing**: Server-side detection and status signaling for 3D/Protected sites.
- [x] **Contextual markers**: Deep-selector identification and element-level annotation.
- [x] **Collaborative Realtime**: Instant peer cursor tracking and live feedback streams.
- [x] **Bridge Extension Logic**: Clean fallback UI for sites requiring the Entrext Lens browser extension.
- [x] **Fullstack Orchestration**: One-click launch via `run_app.py`.

## 🛠️ Tech Stack

- **Frontend**: Next.js 16.2.2 (Turbopack), TailwindCSS, Framer Motion, Zustand.
- **Backend**: FastAPI (Python 3.11), Uvicorn, BeautifulSoup4.
- **Database**: Supabase (PostgreSQL).
- **Styling**: Vanilla CSS Utility Architecture for maximum performance.

## 🏁 Getting Started

Entrext is currently in **MVP (Minimum Viable Product)** phase.

1. Clone the repository.
2. Follow the steps in [SETUP.md](./SETUP.md).
3. Run `python run_app.py` to initialize the fullstack session.
